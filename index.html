Host on GitHub Pages — Step by Step

Get a free permanent URL (e.g. yourusername.github.io/ballpark-diary) that works on any device.

1
Go to github.com and create a free account if you don't have one.
2
Click the + icon (top right) → New repository. Name it ballpark-diary. Leave it Public. Click Create repository.
3
On the new repo page click Add file → Upload files. Drag in your downloaded ballpark_diary.html file.
4
Before uploading, rename the file to index.html — GitHub Pages needs this exact name as the homepage.
5
Scroll down and click Commit changes.
6
Go to your repo Settings → Pages. Under "Branch" select main, folder / (root), and click Save.
7
After ~60 seconds your app is live at https://yourusername.github.io/ballpark-diary. Bookmark it on every device!
8
Moving data between devices: Export JSON on your old device, open the app URL on your new device, use Merge Import. Done.
